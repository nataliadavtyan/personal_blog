More organised code

Making the code readable for other people and for myself in the future is important for working more efficiently.

How to make the code more readable?

As my codes get longer and longer I try to break them into categories. So my CSS would include typography, links, buttons, images, layout and media queries. This helps to quickly identify in which part of the code I need to find each element.

Another thing that I incorporated in this code is also commenting each group of the elements within these categories. If the code is not viewed daily, it is easy to forget where each heading is and what logic in the names of the classes was.

Was is helpful?

It was helpful, yes. I always knew exactly which part of the code I needed to edit. It helped me to save some time and made me less anxious.

But it's important to remember that anyone will be able to read these comments so keep it simple and only about the code. Unless you want a different effect.
